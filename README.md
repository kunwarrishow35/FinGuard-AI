# FinGuard AI 🏦

FinGuard AI is a machine learning-powered loan risk assessment platform that predicts loan approval outcomes using applicant financial and demographic information. The project leverages feature engineering, Random Forest classification, model evaluation, and an interactive Streamlit dashboard to provide real-time loan approval predictions.

## Features

* Loan Approval Prediction
* Random Forest Classifier
* XGBoost Model Comparison
* Feature Engineering
* 5-Fold Cross Validation
* Interactive Streamlit Dashboard

## Tech Stack

* Python
* Pandas
* Scikit-Learn
* XGBoost
* Streamlit
* Joblib

## Model Performance

* Random Forest Accuracy: ~79%
* 5-Fold Cross Validation Accuracy: ~79.3%

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
